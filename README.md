# Using

Just enter the prefix name, and the complete code snippet will be automatically completed.

```html
div →
<div></div>
doc → <!DOCTYPE html>
```

```javascript
imp → import xxx from xxx;
clo → console.log(xxx);
```

```css
col → color: #000;
bg  → background: #fff;
```
